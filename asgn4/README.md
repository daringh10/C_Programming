#Purpose

Encodes bytes via hamming code and correctly outputs
bytes after decoding.


#Build

See to the Makefile

Important Commands:
$make all:
	Executes neccesary makefile commands to create executable 
	
$make gen:
	Creates the generator binary.

$make dec:
	Creates the decoder binary.

$make err:
	Creates the error binary.
	
$make format:
	clang formats the main c files clang style.
	
$make clean:
	cleans out all binary and output files.

	
	
#Run

-Format the main  c files using make format,

-Execute the command $make all to construct
 the binary executables
